# ✅ SEO CHECKLIST UNTUK PORTFOLIO WEBSITE

## 📋 PRE-LAUNCH CHECKLIST

### On-Page SEO

- [ ] Title tag unik dan mengandung keyword utama (50-60 karakter)
- [ ] Meta description menarik dan call-to-action (150-160 karakter)
- [ ] H1 tag digunakan hanya 1x per halaman
- [ ] H2 dan H3 tags terstruktur dengan benar
- [ ] Internal links dengan anchor text yang deskriptif
- [ ] Alt text pada semua images
- [ ] Mobile-responsive design (test dengan Mobile-Friendly Test)
- [ ] Page loading speed < 3 detik (test dengan PageSpeed Insights)
- [ ] SSL Certificate (HTTPS) aktif
- [ ] Canonical URL ditentukan dengan benar

### Technical SEO

- [ ] sitemap.xml ada dan updated
- [ ] robots.txt ada dan benar
- [ ] robots.txt mengizinkan crawler untuk access
- [ ] .htaccess atau server configuration untuk gzip compression
- [ ] Browser caching dikonfigurasi
- [ ] No 404 errors pada internal links
- [ ] No broken external links
- [ ] Structured data (Schema.org) ada dan valid
- [ ] Open Graph tags lengkap
- [ ] Twitter Card tags lengkap

### Content

- [ ] Konten unik dan original (minimal 300 kata per halaman)
- [ ] Keyword naturally di-incorporate dalam content
- [ ] Keyword di H1, first 100 words, dan meta description
- [ ] Content informatif dan user-focused
- [ ] No duplicate content antar halaman
- [ ] Grammar dan spelling sempurna
- [ ] Call-to-action jelas dan prominent

### Performance

- [ ] Images dioptimasi dan di-compress
- [ ] CSS dan JavaScript minified
- [ ] No render-blocking resources
- [ ] Lazy loading untuk images
- [ ] Adaptive images (responsive images)
- [ ] WebP format untuk images (fallback ke PNG/JPG)

### User Experience

- [ ] Navigation intuitif dan mudah
- [ ] Clear page hierarchy
- [ ] Mobile-friendly navigation
- [ ] Touch-friendly buttons (min 44x44px)
- [ ] High contrast untuk readability
- [ ] Font size readable di mobile
- [ ] No intrusive pop-ups (khususnya di mobile)
- [ ] Page load time optimal

### Security

- [ ] HTTPS aktif
- [ ] No malware atau unwanted scripts
- [ ] Password-protected admin area (jika ada)
- [ ] No personal information exposed
- [ ] Form data encrypted
- [ ] Security headers konfigurasi (.htaccess atau server config)

## 🎯 KEYWORD RESEARCH & OPTIMIZATION

### Keyword Strategy

- [ ] Primary keyword untuk halaman (target 1-2 per halaman)
- [ ] Secondary keywords identified
- [ ] Long-tail keywords included
- [ ] Competitor keywords analyzed
- [ ] Search volume vs competition balanced
- [ ] Keywords naturally incorporated

### Example Keywords untuk Portfolio:

```
Primary:
- website designer
- web developer
- portfolio website

Secondary:
- website design services
- freelance web developer
- professional portfolio

Long-tail:
- best website designer in [city]
- how to find a good web developer
- affordable web design services
```
